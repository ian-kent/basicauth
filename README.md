basicauth
=========

Generate HTTP basic authentication tokens from the command line:

```bash
~> go get github.com/ian-kent/basicauth

~> basicauth oauth_token
b2F1dGhfdG9rZW46

~> basicauth username password
dXNlcm5hbWU6cGFzc3dvcmQ=
```
